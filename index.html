Branding Styles Demo

Usage

include the brand.css and disk like this: 
                <link rel="stylesheet" href="https://geba-trans.github.io/settings/geba-bootstrap/dist/css/bootstrap.css">
                <link rel="stylesheet" href="https://geba-trans.github.io/settings/branding.css">
also include the js 
                <script src="https://geba-trans.github.io/settings/geba-bootstrap/dist/css/bootstrap.bundle.min.js"></script>
